# Security Testing of Sample Web Application

## Objective
Identify common web vulnerabilities:
- SQL Injection
- Cross-Site Scripting (XSS)

## Tools
- OWASP ZAP
- Burp Suite Community Edition
- Browser DevTools

## Findings
- SQL Injection (High)
- XSS (Medium)

## Recommendations
- Use prepared statements
- Validate and sanitize input
- Encode output
